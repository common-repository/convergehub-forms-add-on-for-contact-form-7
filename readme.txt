This plugin integrates ConvergeHub with Contact Form 7 forms.

== Description ==

Businesses around the world lose a major chunk of time, manually creating leads. Capture leads from Contact Form 7 and add them to ConvergeHub CRM- WITH JUST ONE CLICK!

This is a FREE plugin which seamlessly integrates ConvergeHub with Contact Form 7 forms.

Once installed and activated, you have to enable the Activate option and fill your details by API credentials. i.e, API Key, API Secret.

After settings up the API you will see the Form Fields option to setup your form fields.

If you have any issues or questions about this plugin then please create a support request to support@convergehub.com, and we will get back shortly.

Note: This plugin uses the PHP library cURL. Please make sure that the cURL library is enabled on your server.

== Installation ==

1. Sign in to your WordPress site as an Administrator.
2. In the main menu go to Plugins > Add New.
3. Search for "ConvergeHub For Contact Form 7" (without the quotes) and click on Install.
4. After the plugin is installed activate the plugin.
5. That's it! Go ahead and start creating forms using Contact Form 7, and you will see a tab called "ConvergeHub Addon".

= What details do I need to start using CF7 form with ConvergeHub? =

= Where can I find those details? =
